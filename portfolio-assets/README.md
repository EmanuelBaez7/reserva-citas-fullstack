# Portfolio Assets

This directory is intended to store high-resolution visual previews of your project for the GitHub presentation layer.

**Recommended Assets:**
- `hero-preview.png` (Landing page hero section)
- `dashboard-user.png` (User dashboard view)
- `booking-flow.png` (Calendar layout or booking modal)
- `auth-flow.png` (Login screen)

*These files are linked directly in the root `README.md`.*
